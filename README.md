# Base Telegram Bot
A pretty simple and flexible boilerplate for making Telegram bot

## Configuration
* BOT_TOKEN
* RULES
* DATABASE_HOST
* DATABASE_PORT
* DATABASE_USERNAME
* DATABASE_PASSWORD
* DATABASE_NAME

## How to install
* npm install
* npm run build

### Greetings

### Squid Game
